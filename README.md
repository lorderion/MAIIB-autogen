Архитектура:

PostgreSQL + pgvector — хранение и семантический поиск по CVE.
FastAPI — асинхронный API для оркестрации.
AutoGen v0.4 — мультиагентная система (планировщик, сканер, аналитик).
Pydantic — валидация данных между компонентами.
Docker — контейнеризация БД и воспроизводимость окружения.
Ключевые особенности:

Гибридный поиск: векторный (семантика) + фильтрация по ключевым словам (точность).
Детерминированные инструменты (Nmap, CVE‑матчинг) вынесены из LLM в Python‑код.
Агенты отвечают только за планирование и генерацию отчётов.
